🎓 CGPA Calculator – Java Swing Project

This project is a simple yet practical CGPA Calculator Desktop Application built using Java Swing.
It is designed to help students quickly calculate their CGPA by entering subject details in an easy-to-use graphical interface.

---

💡 Project Idea

In many cases, students calculate GPA manually which is time-consuming and error-prone.
This application solves that problem by automating the calculation process with a clean GUI.

---

⚙️ Key Functionalities

- 🔐 Student Login System (Name & ID input)
- 📘 Add multiple subjects dynamically
- 📊 GPA calculation based on marks
- 🧮 Accurate CGPA calculation using total credits
- 🖥️ Interactive GUI using Java Swing

---

🧠 How It Works

- The user first enters Name and ID in the login page
- Then inputs:
  - Subject Name
  - Credit
  - Marks
- The system converts marks into GPA using a predefined scale
- Each subject’s weighted GPA is added
- Final CGPA is calculated using:

CGPA = Total Grade Points / Total Credits

▶️ Execution Steps

1. Open the project in NetBeans
2. Run "LoginPage.java"
3. Enter your details
4. Add subjects and press Add
5. Click Calculate CGPA to get result
